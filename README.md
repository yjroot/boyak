# 보약 (Boyak) 프로그래밍 언어

한글 기반의 프로그래밍 언어입니다. Python처럼 간단하고 직관적인 문법을 제공하며, 한국어 문법 구조를 따릅니다.

## 특징

- **한글 키워드**: 모든 키워드와 내장 함수가 한글로 작성
- **SOV 어순**: 한국어의 자연스러운 문장 구조 (주어-목적어-동사)
- **동적 타입**: Python처럼 타입 선언 없이 사용
- **들여쓰기 기반**: 중괄호 대신 들여쓰기로 블록 구분
- **최소 기호**: 가능한 한 자연어에 가까운 표현

## 빠른 시작

### 설치

```bash
git clone https://github.com/your-repo/boyak.git
cd boyak
```

### 실행

```bash
# 대화형 모드 (REPL)
python3 src/boyak.py

# 파일 실행
python3 src/boyak.py examples/01-안녕세상.보약
```

## 기본 문법

### 변수 선언

```보약
이름은 "홍길동"이다
나이는 25이다
키는 175.5이다
활성화는 참이다
```

### 출력

```보약
"안녕하세요"를 출력하라
"이름: {이름}"을 출력하라    # 문자열 보간
```

### 조건문

```보약
만약 나이가 18보다 크면
    "성인입니다"를 출력하라
아니면 나이가 13보다 크면
    "청소년입니다"를 출력하라
그외에
    "어린이입니다"를 출력하라
```

### 반복문

```보약
# 횟수 반복
5번 반복하자
    "안녕!"을 출력하라

# 범위 반복
1부터 10까지 반복하자 숫자로
    숫자를 출력하라

# 컬렉션 반복
과일들은 ["사과", "배", "포도"]이다
각각의 과일을 과일들에서 반복하자
    과일을 출력하라
```

### 함수

```보약
인사를 정의하자 이름을 받아
    "안녕하세요, {이름}님!"을 돌려주라

메시지는 인사("홍길동")이다
메시지를 출력하라
```

### 목록과 사전

```보약
# 목록
과일들은 ["사과", "배", "포도"]이다
첫번째는 과일들[0]이다

# 사전
학생은 {"이름": "홍길동", "나이": 20}이다
이름은 학생["이름"]이다
```

## 설계 철학

보약은 다음 원칙을 따릅니다:

1. **한글 우선**: 한국어 화자가 자연스럽게 읽을 수 있는 코드
2. **SOV 어순 존중**: "X를 출력하라" (목적어 + 동사)
3. **경어법 활용**: -하라(명령), -하자(청유), -이다(서술)
4. **최소 기호**: 자연어에 가까운 표현 선호
5. **가독성 우선**: 작성 편의성보다 읽기 쉬운 코드

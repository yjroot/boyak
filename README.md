# 보약 (Boyak) 프로그래밍 언어

> **뱀탕 (Snake Soup)** - Python으로 만든 보약이니까요! 🐍

한글 기반의 프로그래밍 언어입니다. Python처럼 간단하고 직관적인 문법을 제공하며, 한국어 문법 구조를 따릅니다.

## 특징

- **한글 키워드**: 모든 키워드와 내장 함수가 한글로 작성
- **SOV 어순**: 한국어의 자연스러운 문장 구조 (주어-목적어-동사)
- **동적 타입**: Python처럼 타입 선언 없이 사용 (선택적 타입 힌트 지원)
- **들여쓰기 기반**: 중괄호 대신 들여쓰기로 블록 구분
- **최소 기호**: 가능한 한 자연어에 가까운 표현
- **현대적 기능**: 패턴 매칭, 제너레이터, 데코레이터 등 지원

## 빠른 시작

### 설치

```bash
git clone https://github.com/your-repo/boyak.git
cd boyak
```

### 실행

```bash
# 대화형 모드 (REPL)
python3 src/boyak.py

# 파일 실행
python3 src/boyak.py examples/01-안녕세상.보약
```

## 기본 문법

### 변수 선언

```보약
이름은 "홍길동"이다
나이는 25이다
키는 175.5이다
활성화는 참이다

# 타입 힌트 (선택적)
점수: 정수는 100이다
```

### 출력

```보약
"안녕하세요"를 출력하라
"이름: {이름}"을 출력하라    # 문자열 보간
```

### 조건문

```보약
만약 나이가 18보다 크면
    "성인입니다"를 출력하라
아니면 나이가 13보다 크면
    "청소년입니다"를 출력하라
그외에
    "어린이입니다"를 출력하라
```

### 반복문

```보약
# 횟수 반복
5번 반복하자
    "안녕!"을 출력하라

# 범위 반복
1부터 10까지 반복하자 숫자로
    숫자를 출력하라

# 컬렉션 반복
과일들은 ["사과", "배", "포도"]이다
각각의 과일을 과일들에서 반복하자
    과일을 출력하라
```

### 함수

```보약
인사를 정의하자 이름을 받아
    "안녕하세요, {이름}님!"을 돌려주라

메시지는 인사("홍길동")이다
메시지를 출력하라

# 타입 힌트 포함
더하기를 정의하자 -> 정수 가: 정수와 나: 정수를 받아
    가 + 나를 돌려주라
```

### 목록과 사전

```보약
# 목록
과일들은 ["사과", "배", "포도"]이다
첫번째는 과일들[0]이다

# 사전
학생은 {"이름": "홍길동", "나이": 20}이다
이름은 학생["이름"]이다

# 집합
숫자집합은 {1, 2, 3, 4, 5}이다
```

## 고급 기능

### 클래스와 상속

```보약
사람 틀을 정의하자
    생성할때 이름을 받아
        자신의 이름은 이름이다

    소개를 정의하자
        "저는 {자신의 이름}입니다"를 돌려주라

학생 틀을 정의하자 사람을 상속받아
    생성할때 이름과 학교를 받아
        부모의 생성(이름)
        자신의 학교는 학교이다
```

### 삼항 연산자

```보약
결과는 나이가 18보다 크면 "성인" 아니면 "미성년"이다
```

### 람다 함수

```보약
제곱은 (숫자) -> 숫자 * 숫자이다
결과는 제곱(5)이다  # 25
```

### 리스트 컴프리헨션

```보약
숫자들은 [1, 2, 3, 4, 5]이다
제곱들은 [숫자 * 숫자 각각의 숫자를 숫자들에서]이다
```

### 널 안전 연산자

```보약
# 안전 속성 접근 (?.)
이름은 사용자?.이름이다  # 사용자가 없음이면 없음 반환

# 널 병합 연산자 (??)
이름은 입력값 ?? "기본값"이다  # 입력값이 없음이면 "기본값" 사용
```

### 열거형

```보약
색상 열거형을 정의하자
    빨강
    초록
    파랑

선택은 색상.빨강이다
```

### 패턴 매칭

```보약
값을 맞춰보자
    1이면
        "하나"를 출력하라
    2이면
        "둘"을 출력하라
    그외에
        "기타"를 출력하라
```

### 제너레이터

```보약
숫자생성기를 생성기로 정의하자 최대를 받아
    현재는 0이다
    현재가 최대보다 작을 동안 반복하자
        현재를 내보내라
        현재는 현재 + 1이다
```

### with 문 (컨텍스트 매니저)

```보약
열기("파일.txt", "w")를 파일로 사용하여
    파일.write("내용")
# 자동으로 파일 닫힘
```

### 데코레이터

```보약
@로그출력
인사를 정의하자 이름을 받아
    "안녕, {이름}!"을 돌려주라
```

## 설계 철학

보약은 다음 원칙을 따릅니다:

1. **한글 우선**: 한국어 화자가 자연스럽게 읽을 수 있는 코드
2. **SOV 어순 존중**: "X를 출력하라" (목적어 + 동사)
3. **경어법 활용**: -하라(명령), -하자(청유), -이다(서술)
4. **최소 기호**: 자연어에 가까운 표현 선호
5. **가독성 우선**: 작성 편의성보다 읽기 쉬운 코드

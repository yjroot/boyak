# 보약 프로그래밍 언어 명세서 v1.0

## 개요

**보약**(Boyak)은 한글 기반의 프로그래밍 언어로, 한국어 화자가 자연스럽게 코드를 읽고 쓸 수 있도록 설계되었습니다.

### 핵심 특징

| 특징 | 설명 |
|------|------|
| **한글 우선** | 모든 키워드와 내장 함수가 한글로 작성 |
| **SOV 어순** | 한국어 문법 구조(주어-목적어-동사)를 따름 |
| **동적 타입** | Python처럼 타입 선언 없이 사용 |
| **들여쓰기 기반** | 중괄호 대신 들여쓰기로 블록 구분 |
| **최소 기호** | 가능한 한 자연어에 가까운 표현 |

### 파일 확장자

`.보약` 또는 `.byk`

## 빠른 참조

### 변수와 상수

```보약
# 변수 선언
이름은 "홍길동"이다
나이는 25이다
키는 175.5이다
활성화는 참이다

# 상수 선언
항상 원주율은 3.14159이다
```

### 출력과 입력

```보약
# 출력
"안녕하세요"를 출력하라
"값: {변수}"를 출력하라          # 문자열 보간

# 입력
이름은 "이름을 입력하세요: "로 입력받아라
```

### 조건문

```보약
만약 조건1이면
    # 실행 코드
아니면 조건2이면
    # 실행 코드
그외에
    # 실행 코드
```

### 반복문

```보약
# 횟수 반복
5번 반복하자
    "안녕"을 출력하라

# 범위 반복
1부터 10까지 반복하자 숫자로
    숫자를 출력하라

# 조건 반복
조건인동안 반복하자
    # 실행 코드

# 컬렉션 반복
각각의 항목을 목록에서 반복하자
    항목을 출력하라
```

### 함수

```보약
# 정의
인사를 정의하자 이름을 받아
    "안녕, {이름}!"을 돌려주라

# 호출
메시지는 인사("철수")이다
```

### 데이터 타입

```보약
# 목록
과일들은 ["사과", "배", "포도"]이다
첫번째는 과일들[0]이다

# 사전
학생은 {"이름": "홍길동", "나이": 20}이다
이름은 학생["이름"]이다
```

## 문법 상세

### 1. 어휘 구조

#### 1.1 문자 집합

- 소스 코드는 UTF-8 인코딩
- 한글, 영문, 숫자, 특수문자 사용 가능
- 식별자(변수명, 함수명)에 한글 권장

#### 1.2 주석

```보약
# 한 줄 주석

##
여러 줄 주석
여기에 설명을 작성합니다
##
```

#### 1.3 들여쓰기

- 공백 4칸 또는 탭 1개
- 파일 내 일관성 필수
- 혼합 사용 시 오류

### 2. 데이터 타입

#### 2.1 기본 타입

| 타입 | 예시 | 설명 |
|------|------|------|
| 정수 | `42`, `-17`, `1_000_000` | 정수 |
| 실수 | `3.14`, `-2.5` | 부동소수점 |
| 문자열 | `"안녕"`, `'반가워'` | 텍스트 |
| 불리언 | `참`, `거짓` | 논리값 |
| 없음 | `없음` | null 값 |

#### 2.2 컬렉션 타입

| 타입 | 예시 | 설명 |
|------|------|------|
| 목록 | `[1, 2, 3]` | 순서 있는 수정 가능 |
| 사전 | `{"키": "값"}` | 키-값 쌍 |
| 집합 | `집합(1, 2, 3)` | 중복 없는 집합 |

### 3. 연산자

#### 3.1 산술 연산자

| 연산 | 기호 | 한글 대안 |
|------|------|----------|
| 덧셈 | `+` | `더하기` |
| 뺄셈 | `-` | `빼기` |
| 곱셈 | `*` | `곱하기` |
| 나눗셈 | `/` | `나누기` |
| 정수 나눗셈 | `//` | `몫` |
| 나머지 | `%` | `나머지` |
| 거듭제곱 | `**` | `제곱` |

#### 3.2 비교 연산자

| 표현 | 의미 |
|------|------|
| `같다`, `같으면` | == |
| `다르다`, `다르면` | != |
| `보다 크다`, `보다 크면` | > |
| `보다 작다`, `보다 작으면` | < |
| `보다 크거나같다` | >= |
| `보다 작거나같다` | <= |

#### 3.3 논리 연산자

| 표현 | 의미 |
|------|------|
| `그리고`, `이고` | and |
| `또는`, `이거나` | or |
| `아닌`, `이 아닌` | not |

### 4. 제어 구조

#### 4.1 조건문

```보약
만약 점수가 90보다 크거나같으면
    "A"를 출력하라
아니면 점수가 80보다 크거나같으면
    "B"를 출력하라
그외에
    "C"를 출력하라
```

#### 4.2 반복문

```보약
# 범위 반복
1부터 10까지 반복하자 숫자로
    숫자를 출력하라

# 조건 반복
횟수가 10보다 작은동안 반복하자
    횟수를 출력하라
    횟수 += 1

# 컬렉션 반복
각각의 과일을 과일들에서 반복하자
    과일을 출력하라

# 횟수 반복
5번 반복하자
    "안녕"을 출력하라

# 제어
그만         # break
다음으로     # continue
```

### 5. 함수

```보약
# 기본 함수
인사를 정의하자 이름을 받아
    "안녕, {이름}!"을 돌려주라

# 여러 매개변수
더하기를 정의하자 첫째와 둘째를 받아
    첫째 + 둘째를 돌려주라

# 기본값
인사하기를 정의하자 이름과 인사말을 받아 (인사말은 "안녕")
    "{인사말}, {이름}!"을 돌려주라

# 호출
결과는 더하기(3, 5)이다
인사하기("철수", 인사말="반가워")
```

### 6. 예외 처리

```보약
시도하자
    위험한작업()
오류가나면 값오류로
    "값 오류: {값오류}"를 출력하라
오류가나면 그외오류로
    "오류: {그외오류}"를 출력하라
마무리로
    "정리 작업"을 출력하라
```

### 7. 모듈

```보약
# 가져오기
수학을 가져오라
수학에서 제곱근과 원주율을 가져오라
수학을 ㅅ으로 가져오라

# 사용
결과는 수학.제곱근(16)이다
```

## 내장 함수

### 입출력
- `출력하라(값)` - 화면 출력
- `입력받아라(프롬프트)` - 사용자 입력

### 타입 변환
- `정수(값)` - 정수 변환
- `실수(값)` - 실수 변환
- `문자열(값)` - 문자열 변환
- `목록(값)` - 목록 변환

### 수학
- `절대값(숫자)` - 절대값
- `최대값(값들...)` - 최대값
- `최소값(값들...)` - 최소값
- `반올림(숫자)` - 반올림
- `올림(숫자)` - 올림
- `내림(숫자)` - 내림
- `제곱근(숫자)` - 제곱근

### 컬렉션
- `길이(컬렉션)` - 길이
- `합계(숫자목록)` - 합계
- `평균(숫자목록)` - 평균
- `정렬(목록)` - 정렬
- `뒤집기(목록)` - 뒤집기
- `범위(시작, 끝)` - 범위 생성

## 설계 근거 요약

| 설계 결정 | 근거 |
|----------|------|
| SOV 어순 | 한국어의 자연스러운 문장 구조 존중 |
| 조사 활용 | 한국어 문법의 핵심 요소인 조사로 문법 관계 표현 |

| 들여쓰기 블록 | Python의 성공적 선례, 시각적 명확성 |
| 동적 타입 | 초보자 친화적, 빠른 프로토타이핑 |
| 강타입 | 암시적 변환 방지로 버그 예방 |
| 한글 키워드 | 영어 장벽 제거, 코드 가독성 향상 |
